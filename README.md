# My Trendy Gallery

GitHub Pages로 호스팅하는 심플하고 트렌디한 이미지 갤러리 웹사이트입니다.
**다크 모드 글래스모피즘(Glassmorphism)** 디자인을 적용했으며, 클릭 한 번으로 이미지 URL을 쉽게 복사할 수 있습니다.
바이브 코딩으로 안티그래비티와 제미나이를 활용해서 만들었습니다.
그냥 제가 이미지 url 편하게 웹에서 가져오고 싶은데 날먹하고 싶어서 만든 사이트입니다.

## ✨ 특징
- **반응형 그리드**: 데스크탑과 모바일 어디서나 깔끔하게 보입니다.
- **글래스모피즘**: 세련된 유리 질감과 감각적인 배경 애니메이션을 적용했습니다.
- **편리한 기능**: 이미지를 클릭하면 해당 URL이 즉시 클립보드에 복사됩니다.
- **간편한 관리**: `config.js` 파일만 수정하면 이미지를 쉽게 추가할 수 있습니다.

## 🚀 이미지 추가 방법

1.  **업로드 (Upload)**: `images/` 폴더에 새 사진 파일(예: `photo.jpg`)을 넣습니다.
2.  **등록 (Register)**: `config.js` 파일을 열고 이미지 파일명을 목록에 추가합니다.
    ```javascript
    const galleryConfig = {
      images: [
        "existing.png",
        "photo.jpg" // <--- 이렇게 한 줄을 추가하세요
      ],
      title: "My Trendy Gallery"
    };
    ```
3.  **배포 (Deploy)**: 변경 사항을 GitHub에 저장(Commit & Push)합니다.
